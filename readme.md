# SISTEMA DE GESTAO DE ESTOQUE DA COZINHA

### Descrição do projeto:

O Sistema de Gestão de Cozinha foi desenvolvido com o objetivo de facilitar o gerenciamento das atividades de uma cozinha, permitindo o controle de receitas, ingredientes, estoque e organização diária de forma simples e eficiente.

O projeto foi criado como atividade acadêmica do SENAC, aplicando conceitos de desenvolvimento web, organização de código e boas práticas de programação.

### Funcionalidade:
* Cadastro de clientes
* Login no sistema
* Agendamento de serviços
* Visualização de horários disponíveis
* Confirmação de agendamento
* visualização lista de utensílios



### Tecnologia Utilizadas:
O projeto foi desenvolvido utilizando as seguintes tecnologias:

* HTML
* CSS
* java
* JavaScript
* Spring boot
* GitHub
* Visual Studio Code
* Intelij


### Estrutura do Sistema:
O sistema foi dividido em módulos para facilitar a manutenção e escalabilidade:
* Autenticação de usuários
* Cadastro de clientes
* Gerenciamento de serviços
* Sistema de agendamento
* Painel administrativo


### Benefícios:
Com a implementação deste sistema, a cozinha do SENAC passa a contar com um sistema web que facilitara o cotidiano dos usuários, contendo:

* Organização das informações.
* Agilidade no gerenciamento.
* Melhor controle dos dados.
* Redução de erros em registros.
* Sistema de fácil manutenção.
